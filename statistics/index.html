Requirements

Hardware

  1. Laptop/Desktop - macOS, windows OS, linux OS eg Ubuntu etc. Preferably Windows OS for now.

  2. A second Screen - a monitor, TV, another laptop etc

    • If the second screen is a monitor, it should have a HDMI compatibility. Old ones have VGA input thus you will need to get a HDMI to VGA adapter/converter. If your laptop has USB-C port only then you will need a USB-C to HDMI adapter depending on what your second screen has.

    • For the TV screen, it must be HDMI compatibility. If not, we do not need it.

    • If Laptop, it must satisfy the requirements in 1. above

    • Have a HDMI cable to connect to the second screen from your laptop, if the second screen is any of the first 2 above. -Note that casting is prone to lags hence not recommended.

    • if using old computers, you do not need the HDMI but rather the VGA is sufficient.

  3. A webcam, if not using a laptop with webcam compatibility.

  4. WiFi.

Software

Regarding software, all programs should be downloaded and installed in a folder in a drive separate from the main drive of the computer. Preferably D/E/G if windows. eg C:\online_school\Programs

  1. R & Rstudio: Go to the page RStudio Desktop. Follow the two steps provided on downloading and installing R and Rstudio

  2. Python: Go to the page Anaconda | Anaconda Distribution and download Anaconda.

  3. If windows go to MSYS2 follow the instructions on how to install it.
