Kriging

\[ Y(x) = \mu + Z(x) \]

where

\[ Z(x)\sim \mathcal{N}(0, K) \]

and \(K = \sigma^2R\) is the correlation matrix obtained by any of the well known correlation/kernel functions such as the gaussian, matern, exponential etc.

From the above:

\[ Y\sim N(\mu1, K) \]

The likelihood:

\[ L(h;y) = \frac{1}{(2\pi)^{n/2}|K|^{1/2}}\exp\left(-\frac{(y-\mu1)^\top K^{-1}(y-\mu1)}{2}\right) \]

log-likelihood:

\[ \begin{aligned} l(h;y) =& -\frac{n}{2}\log 2\pi - \frac{1}{2}\log|K| -\frac{1}{2}(y-\mu1)^\top K^{-1}(y-\mu1)\\ =& -\frac{n}{2}\log 2\pi - \frac{n}{2}\log \sigma^2- \frac{1}{2}\log|R| -\frac{1}{2\sigma^2}(y-\mu1)^\top R^{-1}(y-\mu1)\\ \end{aligned} \]

Optimizing Model:

\[ \begin{aligned} \frac{\partial}{\partial\mu}l(h)&:~ 1^\top R^{-1}(y-\mu1) = 0\implies\mu=\frac{1^\top R^{-1}y}{1^{\top}R^{-1}1}\\ \frac{\partial}{\partial\sigma^2}l(h)&: -\frac{n}{2\sigma^2} +\frac{1}{2(\sigma^2)^2}(y-\mu1)R^{-1}(y-\mu1)\\ &\implies \sigma^2 = \frac{1}{n}(y-\mu1)R^{-1}(y-\mu1)\\ \end{aligned} \]

How to compute R? Assume we have \(K\) dimensions. We use the kernel on each dimension independently and then take the product of all the one-dimensional kernels. Note that if the Covariance function is separable, then the resulting kernel value is similar to taking the distance of the whole data. ie \(K(x,y) = K(x)K(y)\) if the kernel is separable. In kriging the notion of separability is assumed even if this is not the case. eg in case of matern function.

Suppose we have

\[ d_k = \theta\frac{|x_{ik} - x_{jk}|}{h_k} \]

where \(h_k\) is the characteristic length scale and \(\theta\) is a rate factor, then we have:

Kernel functions
Kernel R
Gaussian : \(\theta = 1\) \[ R(x_i,x_j) = \prod_{k=1}^K\exp(-d_k^2/2) \]
Matern \(\nu=5/2,\quad\theta=\sqrt{5}\) \[ R(x_i,x_j) = \prod_{k=1}^K\left(1 + d_k + \frac{d_k^2}{3}\right)\exp(-d_k) \]
Matern \(\nu=3/2, \quad\theta=\sqrt{3}\) \[ R(x_i,x_j) = \prod_{k=1}^K\left(1+d_k\right)\exp(-d_k) \]
Exponential \(\theta=1\) \[ R(x_i,x_j) = \prod_{k=1}^K\exp(-d_k) \]
Power Exponential \(\theta=1\) \[ R(x_i,x_j) =\prod_{k=1}^K\exp(-d_k^p/2) \]

Our goal is to estimate the optimal characteristic length scales for each kernel. Use MLE

We differentiate the Likelihood with respect to \(h_k\)

\[ \begin{aligned} \frac{\partial}{\partial h_k}l:=&\frac{\partial}{\partial h_k}-\frac{n}{2}\log 2\pi - \frac{n}{2}\log \sigma^2- \frac{1}{2}\log|R| -\frac{1}{2\sigma^2}(y-\mu1)^\top R^{-1}(y-\mu1)\\ =&-\frac{\partial \operatorname{tr}(\log R)}{2} +\frac{1}{2\sigma^2}(y-\mu1)^\top R^{-1}\partial RR^{-1}(y-\mu1)\\ =&-\frac{\operatorname{tr}(\partial \log R)}{2} +\frac{1}{2\sigma^2}(y-\mu1)^\top R^{-1}\partial RR^{-1}(y-\mu1)\\ =&-\frac{\operatorname{tr}(R^{-1}D)}{2} -\frac{1}{2\sigma^2}(y-\mu1)^\top R^{-1}DR^{-1}(y-\mu1)\\ \end{aligned} \]

where \(D = \partial R\)

We just need to compute \(\partial R\) for each kernel

Differentiating \(d_k\) w.r.t \(h_k\) we get:

\[ \frac{\partial d_k}{h_k} = -\theta\frac{x_{ik} - x_{jk}}{h_k^2}=-\theta \frac{x_{ik}-x_{jk}}{h_k}\frac{1}{h_k} = -\theta \frac{d_{k}}{h_k} \]

Gaussian:

\[ \begin{aligned} \frac{\partial}{\partial h_k}R(x_i,x_j) :=&\partial\prod_{k=1}^K\exp(-d_k^2/2)\\ =&\frac{-2d_kd_k'}{2}\exp(-d^2_k/2)\left[\prod_{i\neq k}^K\exp(-d_i^2/2)\right]\\ =&\theta \frac{d_k^2}{h_k}R(x_i,x_j) \end{aligned} \]

Matern \(\nu=5/2\)

\[ \begin{aligned} \frac{\partial}{\partial h_k}R(x_i,x_j) :=&\frac{\partial}{\partial h_k}\prod_{k=1}^K\left(1 + d_k + \frac{d_k^2}{3}\right)\exp(-d_k)\\ =&\frac{\partial}{\partial h_k}\left(1 + d_k + \frac{d_k^2}{3}\right)\exp(-d_k)\underbrace{\left[\prod_{i\ne k}^K\left(1 + d_i + \frac{d_i^2}{3}\right)\exp(-d_i)\right]}_{R_{-k}}\\ =& \left[\left(d'_k + \frac{2}{3}d_kd'_k\right)\exp(-d_k) -d'_k\left(1 + d_k + \frac{d_k^2}{3}\right)\exp(-d_k)\right]R_{-k}\\ =&\left[\left(d'_k + \frac{2}{3}d_kd'_k\right) -d'_k\left(1 + d_k + \frac{d_k^2}{3}\right)\right]\frac{R}{1 + d_k + \frac{d_k^2}{3}}\\ =&\left[\frac{1 + \frac{2}{3}d_k}{1 + d_k + \frac{d_k^2}{3}} -1\right]d_k'R\\ =&\frac{1 + \frac{2}{3}d_k -1 -d_k - \frac{1}{3}d_k^2}{1+d_k+d_k^2/3} d_k'R\\ =&\theta\frac{(1+d_k)d^2_kR}{3h_k(1 + d_k+d_k^2/3)} \end{aligned} \]

X <- structure(c(4, 2, 4.1, 0.3, 2, 
      5.5, 1.2, 3.7, 2, 2.5), dim = c(5L,2L))
y = c(4.2, 6.1, 0.2, 0.7, 5.2)
n <- length(y)



dst <- Map(\(x)abs(outer(x,x,"-")), data.frame(X)) # x- y

# starting param:
param <- c(1, 2)
one <- rep(1, n)
# Gaussian:
theta <- 1 
gauss <- function(d, h) exp(-(theta*d/h)^2/2)
R <- Reduce("*", Map(gauss, dst, param))
one <- rep(1, n)
mu <- c(one%*%solve(R, y)/one%*%solve(R, one))
var <- c(t(y - mu) %*% solve(R, y - mu)/n)
mvtnorm::dmvnorm(y, mu*one, var*R, log = TRUE)
[1] -11.75485

We can put the above in a function in order to optimize it:

loglk <- function(param, cov_fun, theta){
  R <- Reduce("*", Map(cov_fun, dst, param))
  mu <- c(one%*%solve(R, y)/one%*%solve(R, one))
  var <- c(t(y - mu) %*% solve(R, y - mu)/n)
  mvtnorm::dmvnorm(y, mu*one, var*R, log = TRUE)
}
# to maximize use fnscale = -1
optim(param, loglk, cov_f = gauss, theta = 1, control=list(fnscale=-1))

Note that we did not use the gradient that was initially computed. ie the derivative of the log-likelihood with respect to \(h\) vector. That is because optim function employs a numerical differentiation whenever the gradient function is not passed. We could go ahead and implement the gradient function and the results should be the same as the above. The idea is for faster convergence.

grad_fun <- function(param, cov_fun, theta){
    R <- Reduce("*", Map(cov_fun, dst, param))
    mu <- c(one%*%solve(R, y)/one%*%solve(R, one))
    z <- solve(R, y - mu)
    var <- mean((y-mu)*z)
    g <- numeric(length(param))
    for(i in seq_along(param)){
      d <- theta* dst[[i]]/param[i] # Note that dst is just x-y
      D <- theta *d^2/param[i]*R
     g[i] <-  -sum(diag(solve(R, D)))/2 + z%*%D%*%z/var/2
    } 
   g
}

lower = rep(1e-8, ncol(X))
upper = 2*diff(apply(X, 2, range))


optim(c(4,5), loglk, grad_fun, cov_fun=gauss, theta=1, 
      lower = lower, upper = upper, 
      control = list(fnscale=-1), method = 'L-BFGS')
